============================

The Witch's House ver. 1.07

============================

-----------------------------------------------------
Å° Starting the Game
-----------------------------------------------------
Double-click on "Game.exe" to start the game.

* You must have the RPG Maker VX RTP installed to play.

The RTP can be downloaded from the link below.
http://dl.degica.com/trial/enterbrain/vx/rtp/vx_rtp102e.zip

The default font used by the game is UmePlus Gothic.
If you don't have this font installed, Courier New will be used instead.
Preferably, you should install the font in the UmePlus folder so it can be used.

-----------------------------------------------------
Å° Controls
-----------------------------------------------------
Move Character: Arrow Keys (Up, Left, Right, Down)
* You can also walk diagonally by pressing two directions at once.
Confirm: Space, Enter, Z
Cancel/Show Menu: Escape, Numpad 0, X
Dash: Z, Shift
Return to Title: F12
Save: Talk to black cat
Load: Available from the menu
Settings: F1
Toggle Full Screen: Alt + Enter (simultaneously)
Quit Game: Alt + F4 (simultaneously)
* On the title screen, "Quit" must be selected twice to close the game.

-----------------------------------------------------
Å° Using Items
-----------------------------------------------------
Items are not used automatically.
Stand in front of an object to interact with and use an item from the menu, and it may cause an event.
Some items will have an effect simply due to your having them in your possession.
Check an item's description for more info.
